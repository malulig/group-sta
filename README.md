# Group STA

Group STA is a free, ad-free mobile app for group static apnea training.

## Features
- Group static apnea timer
- Individual timers for participants
- Minute vibration alerts
- Session results
- Multi-language support

## Platforms
- Android (Google Play)
- iOS planned

## Tech stack
- React Native
- Expo
- TypeScript
